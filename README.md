# LiteExport

**LiteExport** is a lightweight, GUI-powered SQLite export tool built with Python and PyQt6.  
It allows you to export tables from `.sqlite` or `.db` files into clean, readable formats such as:

- TXT  
- CSV  
- JSON  
- HTML
